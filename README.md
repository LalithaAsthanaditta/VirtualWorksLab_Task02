# User Profile Card Generator

A simple web application built using **Python Flask, HTML, and CSS** that allows users to create a personalized profile card by entering their name, bio, and profile image URL.

## Features

- Create a user profile card
- Add name, bio, and profile image
- Simple and responsive interface
- Built with Flask framework
- Easy to run and understand

## Technologies Used

- Python
- Flask
- HTML5
- CSS3

## Project Structure

```text
UserProfileCardGenerator/
│── app.py
│── requirements.txt
│── templates/
│   └── index.html
└── static/
    └── style.css
```

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder:
   ```bash
   cd UserProfileCardGenerator
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:
   ```bash
   python app.py
   ```

5. Open your browser and visit:
   ```
   http://127.0.0.1:5000
   ```
